gits

Basics. Not to be professional on this.

  • clone: e.g. to clone this site..
    git clone https://github.com/anylox/anylox.github.io
  • set the remote url: e.g. this site can be set by..
    git remote set-url origin git+ssh://git@github.com/anylox/anylox.github.io
  • status: git status
  • add: git add --all
  • remove: git rm [file name]
  • commit: git commit -m "my message"
  • push: git push -u origin master
