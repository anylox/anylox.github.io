Tips under Chromebook Asus C302ca

Probably works in general. Tested with Asus C302ca

crouton

  • install:
    • direction: here
    • to install, follow 'the easy way'
    • to add xiwi: at the 'shell' (before 'enter-chroot'), run with the '-u' option
    • note: chrootname='xenial'
  • using:
    • ctrl+alt+T. then, enter 'shell'
    • either 'sudo enter-chroot' or 'sudo startxfce4'
    • to run a program via xiwi: xiwi xterm &

dropbox

  • download the deb package from here
  • open X -> package install -> File -> Open -> select the deb file and install
  • in terminal: 'dropbox start -i' to install demon
  • will be directed to a website to link
  • naututil restart message may come. just restart.
