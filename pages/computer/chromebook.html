    • direction: here
    • fix audio: rm /lib/firmware/9d70-CORE-COREBOOT-0-tplg.bin ln -s dfw_sst.bin.sklnau8825max 9d70-CORE-COREBOOT-0-tplg.bin
    • Korean support:
      • Start -> Language Support: add Korean fonts. After a while, it hangs. But, it is ok. Wait about 2 - 3 mins and restart. Seems like working.
      • Left bottom -> Keyboard -> right click -> Input Method Configuration -> unclick 'Only Show Current Langauge' -> type 'Hangul' -> Log out and long in -> Left bottom -> Keyboard -> Input Method -> Select Hangul
      add Korean via Langauge Support

    • Keyboard: Layout -> Chromebook (most models)|Search overlay

  • install:

    • to install, follow 'the easy way'

  • using:
    • ctrl+alt+T. then, enter 'shell'
    • either 'sudo enter-chroot' or 'sudo startxfce4'
    • to run a program via xiwi: xiwi xterm &
